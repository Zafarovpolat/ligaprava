# Лига Права — Юридическая компания

Профессиональный веб-сайт для юридической компании "Лига Права" с современным дизайном, SPA-переходами и интеграцией с 1С-Битрикс.

## Особенности

- ✅ **SPA-переходы:** Плавная навигация без перезагрузки страниц с использованием Barba.js.
- ✅ **Анимации:** Lottie (при первом заходе), круговые переходы между страницами, IntersectionObserver для элементов.
- ✅ **Адаптивный дизайн:** Полная поддержка мобильных устройств и планшетов.
- ✅ **Современный стек:** SCSS, BEM, Vanilla JS + Barba.js.
- ✅ **Битрикс:** Архитектура проекта подготовлена для интеграции с 1С-Битрикс.
- ✅ **Профиль пользователя:** Расширенный профиль с вкладками (ответы, подписки, ограничения) и управлением.

## Установка и разработка

1. **Установка зависимостей:**

   ```bash
   npm install
   ```

2. **Компиляция SCSS:**

   ```bash
   npm run build
   ```

3. **Режим разработки:**

   ```bash
   npm run watch
   ```

4. **Локальный сервер:**
   ```bash
   npm run dev
   ```

## Команды (package.json)

- `npm run build` - Сборка стилей.
- `npm run watch` - Автоматическая сборка при изменениях.
- `npm run dev` - Запуск dev-сервера.
- `npm run build-prod` - Минификация стилей для продакшена.

## Технические детали

### Анимация переходов

Используется **Barba.js** для создания эффекта "бесшовного" приложения. Переходы сопровождаются круговой анимацией оверлея.

### Типографика

Основной шрифт — **Manrope**. Используется переменная в SCSS для управления начертаниями.

### Цвета

- **Основной:** #1D2A45 (темно-синий)
- **Акцентный:** #98885A (золотой)
- **Фон:** #FFFFFF
