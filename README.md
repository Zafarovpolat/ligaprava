# АЛЬТАГРАД LAWYERS - Лендинг

Профессиональный лендинг для юридической компании с современным дизайном и анимациями.

## Особенности

- ✅ Адаптивный дизайн (desktop/mobile)
- ✅ Загрузочный оверлей с анимацией
- ✅ Анимированный текст в hero секции
- ✅ Выверенная типографика (Helvetica Neue)
- ✅ SCSS архитектура
- ✅ SVG логотип
- ✅ Локальные изображения
- ✅ Готов для интеграции с Битрикс

## Структура проекта

```
advocat/
├── index.html          # Главная страница
├── css/
│   ├── style.css       # Скомпилированные стили
│   └── style.css.map   # Source map для отладки
├── scss/
│   ├── style.scss      # Главный SCSS файл
│   ├── _variables.scss # Переменные
│   ├── _mixins.scss    # Миксины
│   ├── _base.scss      # Базовые стили
│   ├── _header.scss    # Стили хедера
│   └── _hero.scss      # Стили hero секции
├── js/
│   └── script.js       # JavaScript функциональность
├── images/             # Папка для изображений
└── package.json        # Зависимости и скрипты
```

## Установка и запуск

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Компиляция SCSS в CSS:**
   ```bash
   npm run build
   ```

3. **Режим разработки (с автокомпиляцией):**
   ```bash
   npm run watch
   ```

4. **Запуск локального сервера:**
   ```bash
   npm run dev
   ```

## Доступные команды

- `npm run build` - Компиляция SCSS в CSS (expanded стиль)
- `npm run watch` - Автокомпиляция при изменениях
- `npm run dev` - Запуск dev сервера с автокомпиляцией
- `npm run build-prod` - Продакшн сборка (минифицированный CSS)

## Анимации

### Загрузочный оверлей
- Показывается при загрузке страницы (2 секунды)
- Плавное исчезновение с переходом к основному контенту

### Hero секция
1. **Появление текста:** Анимация появления слов с задержкой
2. **Скобки:** Рисование декоративных скобок
3. **Выезжание текста:** Через 3 секунды после загрузки текст выезжает в стороны

## Адаптивность

- **Desktop:** Полноэкранный hero с боковым расположением текста
  - Максимальная ширина контейнера: 1920px
  - Отступы по краям: 60px
- **Mobile:** Вертикальная компоновка с объединенным текстом
  - Отступы по краям: 10px

## Цветовая схема

- **Основной:** #1D2A45 (темно-синий)
- **Вторичный:** #979797 (серый)
- **Акцентный:** #98885A (золотой)
- **Фон:** #FFFFFF (белый)

## Интеграция с Битрикс

Проект готов для интеграции с Битрикс:

- Готовые компоненты для переиспользования
- Оптимизированные анимации
